Что такое CSS?

        CSS (Cascading Style Sheets) — это код, который вы используете для стилизации вашей веб-страницы.
Как и HTML, CSS на самом деле не является языком программирования. Это не язык разметки - это язык таблицы стилей. Это означает, что он позволяет применять стили выборочно к элементам в документах HTML. Настройки стилей хранятся в файле с расширением ".css". Для того чтобы подключить файл CSS к Вашему CSS файлу необходимо в шапку сайта (между тегами head /head) вставить следующий тег: link href="styles/style.css" rel="stylesheet" type="text/css" Где значение атрибута href указывает на местоположение файла и его имя.
