Таблицы

тег table

        Тег table служит контейнером для элементов, определяющих содержимое таблицы. 
        Любая таблица состоит из строк и ячеек, которые задаются с помощью тегов tr и td.
    

        Внутри table допустимо использовать следующие элементы: caption, col, colgroup, 
        tbody, td, tfoot, th, thead и tr.
